# Google Calendar API Configuration
# Add these lines to your existing .env.local file

# Google Calendar API Credentials
# Get these from Google Cloud Console: https://console.cloud.google.com/
NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_google_client_id_here
NEXT_PUBLIC_GOOGLE_API_KEY=your_google_api_key_here

# Instructions:
# 1. Go to Google Cloud Console (https://console.cloud.google.com/)
# 2. Create a new project or select existing one
# 3. Enable Google Calendar API
# 4. Create OAuth 2.0 credentials (Web application)
# 5. Add your domain to authorized origins
# 6. Copy Client ID and API Key to this file
# 7. Replace 'your_google_client_id_here' with your actual Client ID
# 8. Replace 'your_google_api_key_here' with your actual API Key
